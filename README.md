
# SAP Quiz App

## Starten

1️⃣ Stelle sicher, dass Node.js installiert ist  
2️⃣ Installiere Abhängigkeiten:
```bash
npm install
```

3️⃣ Starte die App lokal:
```bash
npm run dev
```

## Deployment auf Netlify

1️⃣ `npm run build`  
2️⃣ Lade den `dist`-Ordner bei Netlify hoch oder verbinde ein Git-Repo
